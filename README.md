# 拉一勾
#### 开发一个仿拉勾的App, 基于react-native 0.21，代码用ES6实现，同时支持Android和iOS.

### 安装
```
npm install
```
### 运行
```
react-native run-android

or

react-native run-ios
```

### 依赖项：
```
"dependencies": {
    "react-native": "^0.21.0",
    "react-native-tab-navigator": "^0.2.17",
    "react-native-viewpager": "^0.2.1"
}
```
